# Password-Manager

a password manager GUI app, like my pomodoro app, I first did it in regular tkinter, then i later modernized it with customtkinter and custom message box library.
the GUI app allows a user to enter a website, email/username and password , then the details will be saved into a txt file, in which the user can see the correct details.

also, the app can also generate a password, it is generated by a random combination of letters, numbers and symbols, which are directly added to the clipboard when user clicks "generate password".
and finally the "Add" button adds all the details to the txt file.

after updating the app to catch some errors and also changed a little of the ui, the details will now be stored in a json format instead of plain .txt file, which I think is better for formatting and sharing.
